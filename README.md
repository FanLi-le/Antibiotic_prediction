**本草御菌录 - 中药抗生素成分智能预测系统**  
**简介**  

**本草御菌录**是一款基于深度学习与化学信息学的智能分析工具，专为中药成分的抗生素活性预测设计。它融合了传统中医药理论与现代分子建模技术，帮助研究人员快速筛选中药中潜在的抗菌活性成分，加速药物研发进程。  

### **核心技术**  
1. **Chemprop模型集成**  
   基于PyTorch Lightning框架，采用先进的图神经网络（GNN）架构，支持分子结构（SMILES）的端到端特征学习与抗生素活性预测。  

2. **可视化交互界面**  
   采用PyQt5构建跨平台GUI，提供直观的操作界面，包含：  
   - **单分子预测**：输入SMILES字符串，实时获取抗生素活性评分（绿色标记活性成分）。  
   - **批量预测**：支持CSV文件导入，自动解析SMILES列并生成预测报告。  
   - **模型管理**：灵活加载/卸载预训练模型，实时显示模型性能参数。  
   - **历史记录**：保存预测结果（含时间戳、置信度），支持导出为CSV文件。  

3. **数据可视化与反馈**  
   - 预测结果以颜色编码（绿色：活性，黄色：非活性，红色：失败）直观展示。  
   - 日志面板实时记录操作流程与异常信息，便于调试与追踪。  

### **应用场景**  
- **中药现代化研究**：挖掘传统草药中的新型抗菌成分，辅助药物靶点发现。  
- **抗菌药物开发**：快速筛选天然产物库，降低实验成本。  
- **教学与科研**：提供可视化分析工具，助力中医药与化学交叉学科教学。  

### **使用优势**  
- **高效便捷**：一键式预测流程，无需复杂编程。  
- **精准可靠**：基于大规模分子数据训练的Chemprop模型，预测结果可解释性强。  
- **扩展性强**：支持自定义模型路径，兼容多种分子表征格式。  

**启动指引**  
1. 运行程序后，通过“模型管理”加载预训练权重（`.ckpt`或`.pt`文件）。  
2. 在“预测”标签页输入SMILES或导入CSV文件，点击“预测”获取结果。  
3. 查看历史记录或导出数据，支持进一步分析。  

**示例输入**  
```python  
# 阿司匹林（非抗生素）  
CC(=O)OC1=CC=CC=C1C(=O)O  

# 青霉素（抗生素）  
CC1(C)SNC(=OCc3ccccc3)C(=O)N2[C@H]1C(=O)O  
```
